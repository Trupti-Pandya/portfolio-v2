01 · MODEL CHIP
Inline selector in the input bar
A small clickable chip sits bottom-left of the input — shows the active provider dot + model name. Exactly how Cursor and Claude.ai do it. One click opens the picker.
02 · PICKER POPUP
Floats above the input
A compact dropdown lists all three providers with model name, region, and cost. Checkmark shows active. Click any to switch — the chip updates instantly, next message uses the new model.
03 · PROVIDER DOT
Colour-coded at a glance
Azure = blue glow. AWS = orange glow. GCP = purple glow. The dot in the chat bubble header and in the chip always match — visitor always knows which hyperscaler answered.
